## 🌐 Overview

Website portofolio pribadi yang dibangun dengan **React + Vite** sebagai frontend dan **Laravel** sebagai backend REST API. Seluruh konten bersifat **dinamis dan real-time** — semua data ditarik langsung dari API sehingga perubahan konten bisa dilakukan kapan saja tanpa perlu menyentuh kode frontend sama sekali.

Desain mengusung tema **dark mode modern** dengan nuansa warna gelap yang elegan, animasi halus, dan tampilan yang responsif di semua ukuran layar.

---

## ✨ Fitur Utama

🎨 &nbsp;**Dark Mode Design** — Tampilan gelap modern dengan aksen warna cyan dan purple

⚡ &nbsp;**Dynamic Content** — Semua konten diambil secara real-time dari Laravel REST API

📱 &nbsp;**Fully Responsive** — Tampil sempurna di desktop, tablet, maupun mobile

🔧 &nbsp;**Skills Section** — Menampilkan skill dengan progress bar dan filter berdasarkan kategori

💼 &nbsp;**Projects Showcase** — Galeri project lengkap dengan tech stack, demo link, dan GitHub link

📅 &nbsp;**Experience Timeline** — Riwayat pengalaman kerja dan pendidikan dalam tampilan timeline

🛡️ &nbsp;**Admin Dashboard** — Panel admin full CRUD untuk mengelola seluruh konten portofolio

---

## 🛡️ Admin Dashboard

Dilengkapi dengan **Admin Panel** berbasis Laravel Blade + Tailwind CSS yang bisa diakses secara terpisah. Admin dapat mengelola seluruh konten portofolio secara penuh tanpa perlu menyentuh kode, meliputi:

- 👤 **Profil** — Edit nama, tagline, deskripsi, foto profil, CV, dan link sosial media
- 🔧 **Skills** — Tambah, edit, hapus skill beserta level proficiency dan kategori
- 💼 **Projects** — Kelola project dengan upload gambar, tech stack, demo & GitHub link
- 📅 **Experiences** — Atur riwayat pengalaman kerja dan pendidikan

Autentikasi admin menggunakan **Laravel Sanctum** untuk keamanan akses panel.
